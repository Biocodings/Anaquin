  General options
 
       <tool>
            Execute a data analysis tool.

       -v
            Display version information.

       -o <directory> = output
            All output files will be saved in the directory. The default is "output".
            
       -h help usage
            Display help usage information.

  Analysis tools

       Usage: anaquin <tool> [options]
