Anaquin (Tools for sequin data analysis) - www.sequins.xyz

Version: 1.1.1

  Please visit our website for more information. Developed by Ted Wong & Tim Mercer, Garvan Institute of Medical Research.

  General options
 
       -t <tool>
            Execute a data analysis tool.

       -v
            Display version information.

       -o <directory> = output
            All output files will be saved in the directory. The default is "output".

  Analysis tools

       Usage: anaquin -t <tool> [options]
