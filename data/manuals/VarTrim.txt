<b>ANAQUIN MANUAL</b>

<b>NAME</b>
     VarTrim - Trim artificials reads aligned to the sequin boundaries

<b>DESCRIPTION</b>
     Sequin alignments give artificials near the region boundaries. This tool can be used to trim those reads.

<b>SUPPORT SOFTWARE</b>
     Users are required to provide alignment file (SAM/BAM) generated by short-read aligner, such as Bowtie and BWA.
     
<b>USAGE EXAMPLE</b>
     anaquin VarTrim –rbed reference.bed –meth leftRight -useqs sequins.bam

<b>TOOL OPTIONS</b>
     Required:
        -rbed                Annotation file (BED format) of mirror regions used for sub-sampling.
        -method = leftRight  How to trim artificials reads aligned to the sequin boundaries. Possible values include left, right and leftRight.
        -useqs               Generated alignments for sequins in SAM/BAM format 

<b>OUTPUTS</b>
     Trimmed alignments are directly written to the console. Users are recommended to pipe outputs to a new file. For
     example, the following command pipes the outputs to the BAM format:
        
          anaquin VarTrim –rbed reference.bed –meth leftRight -useqs sequins.bam | samtools view -bS - > trimmed.bam

     VarTrim_summary.stats - provides summary statistics
